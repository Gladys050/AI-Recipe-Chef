🍳 AI Recipe Chef

 An AI-powered cooking assistant that turns your ingredients into smart, personalized recipes in seconds.

🔗 Live App: https://ai-recipe-chef-five.vercel.app/
💻 Source Code: https://github.com/Gladys050/AI-Recipe-Chef

🚀 Overview

AI Recipe Chef is a full-stack-inspired web application that uses artificial intelligence to generate structured, easy-to-follow recipes based on user input such as dish ideas, available ingredients, and dietary preferences.

The goal of this project is to demonstrate:

Real-world AI integration
Clean UI/UX design
Frontend engineering best practices
Problem-solving through product thinking

🧠 How It Works
User enters:
Dish name
Optional ingredients
Dietary preference
App constructs a structured prompt:
Includes formatting instructions for consistent HTML output
AI processes the request and returns a formatted recipe
App:
Cleans response
Dynamically renders structured UI
